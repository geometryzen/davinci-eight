AttribLocation Class

Defined in: src/davinci-eight/core/AttribLocation.ts:40
Module: EIGHT

Utility class for managing a shader attribute variable. Convenience class that assists in the lifecycle management of an atrribute used in a vertex shader. In particular, this class manages buffer allocation, location caching, and data binding. While this class may be created directly by the user, it is preferable to use the AttribLocation instances managed by the Program because there will be improved integrity and context loss management.
